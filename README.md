# TextCase

A full-stack text-based Computer-Aided Software Engineering (CASE) tool.

## Features

- [ ] Requirements management
- [ ] Test case management
- [ ] Modeling support
- [ ] Text-based interface
- [ ] Web interface
- [ ] Command-line interface

## Installation

```bash
pip install -e .
```

## Usage

```bash
tse
```

## Development

```bash
# Install development dependencies
pip install -e ".[dev]"

# Run tests
pytest
```
